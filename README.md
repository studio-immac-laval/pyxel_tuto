## Sommaire

1. Déplacer un carré/sprite avec les touches du clavier
2. Déplacer un carré/sprite automatiquement vers un autre carré/sprite
3. Accélérer ou ralentir les déplacements
4. Ajouter des interactions entre les carrés/sprites (détecter les collisions)
5. Faire sauter un carré/sprite
6. Faire défiler l'arrière plan (scrolling)
7. Créer une trajectoire parabolique
8. Gérer un projectile
9. Créer un brouillard de guerre/halo lumineux
